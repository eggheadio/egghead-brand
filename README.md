# egghead-brand

egghead's brand hub — the source of truth for the visual language, in plain
paired HTML mockups you can open in a browser.

## What's here

- **`brand.html`** — the brand hub: open this first. It frames every section
  (foundations, components, navigation, footer, resource list, article, full
  layout) as day-shift / night-shift previews.
- **Kit pages** — each surface as a standalone light/dark pair:
  `components.html`, `navigation.html`, `footer.html`, `resource-list.html`,
  `layout.html`, and `article.html` (`*-dark.html` for night shift).
- **`article.html` / `article-dark.html`** — the **branded article reference**:
  a single reading column that breaks out for figures and code. Copy it to
  start a new long-form piece.
- **`assets/`** — the curated art the pages render, including `assets/logos/`
  (egghead logos & identity marks).
- **`EGGO.md`**, **`SOUL.md`**, **`newegghead-research.md`**, **`AGENTS.md`** —
  the mascot spec, brand personality, research, and agent notes.
- **`references/`** — folder map for the raw source art (the art itself lives on
  Dropbox, not in git — see `references/README.md`).

## Conventions

Nunito for everything, Caveat for handwritten margin notes. Paper-and-yolk
palette by day, the same tokens inverted by night; paper objects (sticky notes,
index cards) keep daylight colors even in dark mode. Drawn, not rendered — the
egg does not crack. 🥚
